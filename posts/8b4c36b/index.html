美化 Gnome 桌面

前言

本文基于 Ubuntu 22.04 LTS

Gnome 拓展

安装 tweaks：

安装 gnome-shell-extensions


在浏览器拓展商场中安装 GNOME shell 集成, chrome 和 firefox 都支持。

以 chrome 例，打开 chrome 应用商场。安装后就可以对扩展进行添加以及管理了。

拓展可以自行去 https://extensions.gnome.org/ 寻找。以下是我安装的拓展
